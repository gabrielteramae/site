Página web desenvolvido - Em desenvolvimento

O Reader Books é uma página web desenvolvida para reunir e recomendar livros em três categorias essenciais para quem deseja se desenvolver na área de tecnologia e produtividade pessoal:

🧠 Lógica de Programação — livros que ensinam a entender e resolver problemas de lógica na programação

🔄 Hábitos que Mudam sua Rotina — leituras sobre produtividade, disciplina e mudança de comportamento

💻 Linguagens de Programação — guias práticos sobre linguagens diversas existentes

🛠️ Tecnologias Utilizadas
  - HTML5
  - CSS3
  - JavaScript
